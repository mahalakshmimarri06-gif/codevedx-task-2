# Password Generator & Strength Checker
A modular, security-focused Python tool designed to generate cryptographically secure passwords and evaluate their vulnerability level.

## Features
- **Well-Structured:** Logic separated into distinct functions.
- **Security-Focused:** Mandatory inclusion of uppercase, lowercase, numbers, and special characters.
- **Reproducible:** Functions can be imported into other security tools.
- **User-Friendly:** Interactive loop for continuous generation.

## How to Run
1. Install Python.
2. Run `python main.py` in your terminal.
